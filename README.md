# LogisticRegression

This repository showcases my work on predicting Claimants who come with attorney, using a claimants dataset. The project flow involves,

Data Collection and Exploratory Data Analysis (EDA): I started by gathering the dataset and performing EDA to understand the data’s structure and key features.

Model Building with Train-Test Split: I built model using a train-test split and also followed stratified sampling

Evaluation Metrics: After building the models, I evaluated their performance using metrics like confusion matrix, accuracy score, classification report as the output variable here is "discrete" 

Predictions and Insights: I used ROC_CURVE and AUC to understand how my model would behave in real-time for True-Positive predictions
